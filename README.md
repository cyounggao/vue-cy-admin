# 一个后台管理系统模板

## 启动项目

```bash
# 安装依赖
npm install

# npm 下载速度慢的问题可以修改npm源后再 npm install
npm config set registry https://registry.npm.taobao.org

# 启动开发环境
npm run dev

默认用户
admin
123456
```

## 打包生产

```bash
# 测试环境
npm run test

# 生产环境
npm run build
```

## 其他

```bash
# 代码格式检查，并自动修复
npm run lint
```
